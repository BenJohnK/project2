  • HTML is a language for describing web pages.

    • Stands for Hyper Text Markup Language
    • Describes the structure of Web pages using markup

Elements

Attributes

  • All HTML elements can have attributes.

    HTML 'canvas' and ' SVG' Element

  • It is a container for canvas graphics.
  • An HTML canvas is a rectangular area on an HTML page.
  • Canvas has several methods for drawing paths, boxes, circles, text, and graphic images.
  • svg stands for Scalable Vector Graphics
  • svg is used to define graphics for the Web
  • svg element is a container for SVG graphics.
